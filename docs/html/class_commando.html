Detailed Description

Klasa reprezentująca komandosa, dziedzicząca po Agencie.

Komandos może przechowywać przedmioty (inventory) i wchodzić w interakcje z otoczeniem poprzez nadpisaną metodę doSpeak(). Klasa zarządza pamięcią za pomocą wskaźników smart (unique_ptr).
