Detailed Description

Główna klasa odpowiadająca za przebieg symulacji misji.

Zarządza stanem misji, interakcjami między komandosami, strażnikami i zakładnikami, a także obsługuje efekty specjalne (flashbang, smoke) i zapis logów z przebiegu.
