Detailed Description

Abstrakcyjna klasa bazowa dla przedmiotów w symulacji.

Każdy przedmiot posiada nazwę, może być użyty tylko raz i wywiera efekt na misję poprzez metodę applyEffect(). Stan użycia jest przechowywany i dostępny do modyfikacji.
