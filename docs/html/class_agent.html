Detailed Description

Bazowa klasa reprezentująca agenta w symulacji.

Agent może być "żywy" lub "martwy", może wykonywać akcje oraz wchodzić w interakcje z innymi obiektami. Stan agenta jest kontrolowany przez flagę alive. Klasa zawiera też wirtualną metodę do komunikacji (speak), możliwą do nadpisania.
