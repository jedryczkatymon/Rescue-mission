Detailed Description

Przedmiot typu granat hukowy używany w misji.

Flashbang jest specjalnym przedmiotem, który wpływa na misję, najprawdopodobniej tymczasowo dezorientując przeciwników lub wpływając na inne mechaniki. Dziedziczy po klasie Item.
